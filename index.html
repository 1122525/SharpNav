SharpNav

Advanced Pathfinding for C#

Download this project as a .zip file Download this project as a tar.gz file

What is it?

SharpNav is a library that generates, saves, loads, and finds paths through navigation meshes. It is functionally equivalent to Mikko Monomen's wonderful Recast Navigation.

Note: SharpNav is still in the early stages of development, a lot of features have yet to be implemented.

Platforms

Upon release, SharpNav is planning on integrating with the following platforms (in addition to a standalone release):

  • XNA/MonoGame
  • Unity3D
  • OpenTK
  • SharpDX

As well as any other 3d game/graphics platforms that are requested.
